
# Newsletter Sign-Up Form with Success Message

This is my solution to the [Frontend Mentor Newsletter sign-up form with success message challenge](https://www.frontendmentor.io/challenges/newsletter-signup-form-with-success-message-3FC1AZbNrv).

## Project Overview

This project is a responsive newsletter sign-up form that validates user input and displays a custom success message. The goal was to match the provided design as closely as possible for both desktop and mobile.

## Screenshot

![Project Screenshot](.project-output.png)

## What I Did

- Built the layout using semantic HTML and CSS, focusing on accessibility and responsive design.
- Used Flexbox and CSS Grid for layout, and custom properties for colors and spacing.
- Implemented form validation and dynamic error/success messages with JavaScript.
- Matched the design for both mobile and desktop, including hover/focus states and error handling.

## What I Learned

- How to structure forms for accessibility (using labels, aria attributes, and error messages).
- Responsive design techniques, including mobile-first CSS and media queries.
- The difference between Flexbox and Grid, and when to use each.
- How to debug CSS specificity and layout issues using browser DevTools.
- The importance of matching design details (spacing, alignment, button states) for a polished UI.

## Built With

- Semantic HTML5 markup
- CSS custom properties
- Flexbox & CSS Grid
- Mobile-first workflow
- Vanilla JavaScript
